Одиночка (Singleton)

Гарантирует, что некоторый класс может иметь только один экземпляр, и предоставляет глобальную точку доступа к нему.

Применимость:

-должен быть ровно один экземпляр некоторого класса, легко доступный всем клиентам;
-единственный экземпляр должен расширяться путем порождения подклассов, и клиентам
нужно иметь возможность работать с расширенным экземпляром без модификации своего кода.
